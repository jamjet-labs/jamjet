# Architecture Decision Records

ADRs document significant architectural decisions: why they were made, what alternatives were considered, and what the consequences are. ADRs are **never deleted** — superseded ADRs are marked as such and linked to their replacement.

## Format

JamJet uses the [MADR](https://adr.github.io/madr/) format.
