# Audit export size

Measures the record size and storage cost of a JamJet audit trail across realistic workloads.

**JamJet-specific hypothesis (pending validation):** an audit record per tool call is expected to add bounded overhead suitable for production retention windows. Final numbers depend on the methodology and benchmarks to be published as evidence is ported.

Methodology and reproduction instructions to follow as benchmarks are ported from `jamjet-labs/jamjet-benchmarks`.
